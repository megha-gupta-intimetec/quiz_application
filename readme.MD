How to Run the Project



1 Clone the Repository
git clone <your-github-repo-url>
cd Quiz_Application


2️ Install TypeScript (if not installed)
npm install -g typescript


3️ Compile TypeScript
npx tsc
This will generate the JavaScript file inside the dist/ folder.


4️ Run the Project in Browser
Do NOT run node dist/script.js
Instead, open the project using a browser:
Open index.html directly
OR
Use Live Server (recommended)
If using VS Code:
Install Live Server extension
Right-click index.html

Click Open with Live Server
